# TN Court Hybrid VC Links — PWA

Find hybrid video conferencing links for all 38 Tamil Nadu district courts instantly.

## Files in this folder

| File | Purpose |
|---|---|
| index.html | Main app |
| manifest.json | PWA config (name, icons, colors) |
| sw.js | Service worker (offline support) |
| icon-192.png | App icon (small) |
| icon-512.png | App icon (large) |

## Step 1 — Upload to GitHub Pages

1. Go to https://github.com → Sign up / Log in
2. Click "+" → "New repository"
3. Name: `tn-vc-courts` → Public → Create
4. Click "uploading an existing file"
5. Drag and drop ALL 5 files above → Commit
6. Go to Settings → Pages → Branch: main → Save
7. App is live at: https://YOUR-USERNAME.github.io/tn-vc-courts

## Step 2 — Convert to Android APK

1. Go to https://pwabuilder.com
2. Enter your GitHub Pages URL → Start
3. Click "Package for stores" → Android
4. Download → you get a .aab file

## Step 3 — Publish to Play Store

1. Go to https://play.google.com/console
2. Pay $25 one-time fee
3. Create app → Upload .aab file
4. Fill details → Submit for review (3-7 days)

## App Details for Play Store

- App name: TN Court Hybrid VC Links
- Short description: Instant hybrid VC links for all 38 TN district courts
- Category: Tools / Legal
- Content rating: Everyone
